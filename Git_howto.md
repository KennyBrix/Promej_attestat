# Подсказка по GIT

Создание репозитория:
```sh
git init
```
Для добавления всех изменений в вашем файле:
```sh
git add имя файла 
```
Создание коммита:
```sh
git commit -m "Message"
```
Просмотреть все созданные коммиты:
```sh
git log
```
Просмотреть все созданные коммиты в сокращённом виде в одну линию:
```sh
git log --oneline
```
